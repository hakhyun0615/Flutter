# 클래스의 생성자
``` dart
class Person{
  // 멤버변수
  String name; 
  int age;
  String sex;
  
  // 생성자
  Person(String name, int age, String sex){ 
    this.name = name;
    this.age = age;
    this.sex = sex;
  }
}

void main() {
  // 같은 클래스를 통해서 생성된 인스턴스이지만 생성자에 따라 서로 다른 존재가 되어버림
  Person p1 = new Person('Tom', 30, 'male');
  Person p2 = new Person('Jane', 27, 'female'); 
  print(p1.age);
  print(p2.age);
}
```

# named argument
* argument가 optional하게 됨
``` dart
class Person{
  // 멤버변수
  String name; 
  int age;
  String sex;
  
  // 생성자
  Person({String name, int age, String sex}){ // named argument
    this.name = name;
    this.age = age;
    this.sex = sex;
  }
}

void main() {
  Person p1 = new Person(sex: 'female', age: 30);
  Person p2 = new Person(); 
}

```
